ml_libs
=======

此repo的主要作用有两个：
1. 收集整理常用的机器学习开源代码，以及维护我们自己开发的代码；
2. 统一比较不同算法在各种类型数据集上所表现出来的不同效果，可以对各个算法的适用情况充分了解

目录结构如下：
---------------------
./src 我们实现的机器学习算法的源代码 <br />
./thirdparty 我们从互联网收集的工具源代码，所对应的源代码库作为子目录名称 <br />
./docs 机器学习的相关paper，建议按照不同的主题进行组织，比如classify,regression,...<br />
./data 收集整理标准数据集，以及我们在实验中是如何使用的 <br />
./experiment 将不同机器学习算法在各种类型的数据上做统一比对，实验结果长期积累 <br/>
